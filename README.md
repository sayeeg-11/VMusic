## Quick Setup (Local Development)

```sh
git clone https://github.com/mr-baraiya/VMusic.git
cd VMusic
npm install
cp .env.example .env
npm run dev
````

Update `.env` with:

* Jamendo Client ID
* Spotify Credentials
* Firebase Config
* EmailJS Details
